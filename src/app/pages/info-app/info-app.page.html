Information Explocation

Explocation is an app where the user can discover and know some city places while he answers questions about places that he visits. The objective of the app is the user know more about different city places around there. In the beginning, the app shows a map with markers that represents some places. The user clicks the mark that he wants and then appear a new mark with his location and a line between the new mark and the clicked mark than represent the distance between the user and the place. Down the map, the app shows the distance in meters, and down the meters, appear an image what is not complete. The user needs to approach the place to obtain the full image. When the user is close enough to the place, the image disappears and appear a question about the place instead.
